# Instagram Embed Mendix Pluggable Widget

Instagram Embed Mendix Pluggable Widget based on
[https://www.npmjs.com/package/react-social-media-embed](https://www.npmjs.com/package/react-social-media-embed/) that
allows you to "(...) easily embed social media posts from (...) Instagram (...)". Check how it works
[here](https://github.com/justinmahar/react-social-media-embed?tab=readme-ov-file#instagram-1).
